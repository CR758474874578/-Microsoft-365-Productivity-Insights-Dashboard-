# Microsoft 365 Productivity Insights Dashboard  

📊 **A Data-Driven Analysis of Workplace Productivity & Collaboration using Microsoft 365 Data**  

This project analyzes employee collaboration activities (Meetings, Emails, Chats, Documents) and provides actionable insights through **SQL + Power BI**.  

---

## 🚀 Project Overview  
- Developed a **Productivity Insights Dashboard** using Microsoft 365 dataset.  
- Cleaned & processed data using **MySQL**.  
- Created interactive **Power BI reports** to visualize productivity trends.  
- Derived **Productivity Scores** for employees, departments, and locations.  

---

## 📂 Data Sources  
- **Employees Data** – employee profiles, departments, locations  
- **Meetings Data** – meetings count & trends  
- **Emails Data** – email communications volume  
- **Teams Chats Data** – collaboration & instant messaging  
- **Documents Shared Data** – shared files, docs, and knowledge exchange  

---

## 🔑 Key Features  
- **KPIs**: Total Meetings, Emails, Chats, Docs Shared  
- **Monthly Meeting Trends**  
- **Top 10 Busiest Employees**  
- **Department-wise Collaboration Score**  
- **Location-wise Productivity Analysis**  
- **Employee Productivity Score Calculation**  

---

## 🛠️ Tech Stack  
- **Database**: MySQL  
- **Visualization**: Power BI  
- **Languages**: SQL, Python (for data generation & preprocessing)
